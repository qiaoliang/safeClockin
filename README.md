# 安卡好习惯

这是一个打卡微信小程序。小程序的目标人群是独居者（有老人，也可能是workalone 的中年人）。其目的是以轻量级的介入方式，为社区工作人员（或者独居者的亲友）监护独居者的安全提供便利。核心功能如下：

1. 用户每天为其活跃度打卡。比如，每天一次。这样，一旦某一天（或连续两天）没有打卡，社区人员（或亲友）就会得到通知，该独居用户可能存在风险。可能通过发短信的方式提醒用户打卡。在规定时间内，仍旧没有打卡，需要打电话联系用户。以确认其安全。
2. 每个用户可以为不同的事项设置打卡的规则。比如，吃药提醒；锻炼提醒等。
3. 用户可以通过微信好友或好友的电话来邀请其他人来监督自己打卡。被邀请者接受后，就可以看到邀请者的打卡记录。如果邀请者没有及时打卡，根据邀请者设定的规则，被邀请者就会得到站内消息的提醒。
4. 用户也可以申请成为其它用户的打卡监督人。需要被申请者同意，方可成为监督人。
5. 用户需要注册和登录。注册可以使用微信或电话号码完成。当需要成为其它的人打卡监督人时，必须提供正确的电话号码，以便能及时收到通知。

# 开发环境与技术框架

1. 使用 uni-app + vue3 + pinia + scss + typescript 和 uview-plus 开发。 
2. 如果不必要，也可以使用 uni-app 的内部组件和扩展组件。
3. 如果需要使用除上面提到的组件外的其它组件或库文件，需要停下来，询问用户是否允许使用额外的组件或库文件。
